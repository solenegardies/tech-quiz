# Web rules

- **One component per file**.
- **Use Tailwind utility classes**: No hardcoded colors or spacing outside of Tailwind.
- **Use i18n keys for all user-facing text**: No hardcoded strings — use `useTranslation()` from `lib/i18n`.
- **Server/client split**: Pages (`page.tsx`) are server components. Interactive logic goes in `'use client'` companion components (e.g., `DashboardClient.tsx`).
